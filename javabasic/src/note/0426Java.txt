### 진법변환

# 진법 : 수의 묶음 단위
   10진법 : 0~9까지 수를 사용해서 수를 묶은 단위
   2진법 : 0~1까지 수를 사용해서 수를 묶은 단위 => 0
   8진법 : 0~7까지 수를 사용해서 수를 묶은 단위 => 000
   16진법 : 0~F까지 수를 사용해서 수를 묶은 단위 => 0000
   * 10 => 2진수 => 10을 2로 계속 나누어서 나머지를 역순 정리
   * 2진수 => 10진수 => 각 자리수의 합을 구하면 됨

### 로직 (Logic)

# 프로그래밍 세계의 로직은 딱 2가지 : 분기, 반복

 * 분기
  - if ~ else if ~ else
  - switch ~ case ~ default

 * 반복
 
  - for
    for (초기값; 조건식; 증감식) {
	반복적으로 수행할 구문들...
    }
    1. 초기값은 for문에서 처음에 한번만 수행
    2. 초기값으로 조건식을 수행해보고 결과가 true이면 구문들을 수행
    3. 증감식을 수행
    4. 조건식을 수행해보고 결과가 true이면 구문들을 수행
    5. false이면 for의 수행을 끝냄
    
  - for : (enhanced for, 향상된 for문)
    배열, List ... 여러값들을 가진 타입들의 각 값을 하나씩 꺼내서 반복시킬때 사용
  
  - while : 조건이 true인 동안 계속 반복
  
  - do while : 한번 do구문을 수행하고 조건이 true인 동안 계속 반복
  
  - break : 가장 가까운 반복문을 즉시 빠져나감
  
  - continue : 가장 가까운 반복문의 다음 반복을 수행
