### JDBC 실습 0523

[학생정보관리프로그램]

# 계정 : 이니셜계정 사용

# 데이터
1. 학생 : 학번, 학생명, 나이, 성별, 과목
2. 과목 : 과목번호, 과목명

# 프로그램
1. 학생테이블, 과목테이블, 학생시퀀스, 과목시퀀스를 Statement로 생성
2. 학생테이블과 과목테이블에 PreparedStatement로 5개씩 데이터를 입력
3. 학생정보와 과목정보를 아래와 같이 출력
    1번 홍길동 : 20살, 남자, 국어/영어/수학
    2번 김길순 : 30살, 여자, 국어/영어
    3번 최길동 : 40살, 남자, 영어/수학
    4번 박길순 : 50살, 여자, 국어/수학
    5번 유길동 : 60살, 남자, 국어/영어/수학/과학
