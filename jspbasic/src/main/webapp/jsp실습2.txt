# jsp실습2 : 퀴즈프로그램

* 폴더 : quiz
* 파일 : quiz1.jsp ~ quiz5.jsp, quizResult.jsp
* 기능 : quiz1 ~ quiz5까지 각각 한 문제씩 출제 맞춘 점수를 quizResult에서 출력
* 예시
  quiz1.jsp : 세계에서 가장 가난한 왕은? 최저임금
  quiz2.jsp : 왕이 넘어지면? 킹콩
  quiz3.jsp : 높은 곳에서 아이를 낳으면? 하이에나
  quiz4.jsp : 자가용의 반대말은? 커용
  quiz5.jsp : 청바지를 땅에 심으면? 심은진
  quizResult.jsp : 5/5 문제 정답~
